# OUTPUT FORMAT (Chain of Thought)
Don't rush to code. Think step-by-step.
1. Wrap your analysis and logic in <thought> tags.
2. Wrap the final executable code in <code> tags.
3. Return ONLY these tags. No other markdown.

Example:
<thought>
The user wants to analyze X. The table Y has columns A and B. I should aggregate...
</thought>
<code>
SELECT ...
</code>
